Aplicativo de projetos de pesquisa: 
- NPM init para baixar os packages do projeto 
- Baixar android studio
- Baixar JDK 8.x
- Baixar Gradle
- ionic cordova
- comandos para execução do projeto: 
  - cordova.cmd build android --release;
  - ionic cordova build android;
  - cordova run android --emulator (Caso não possua dispositivo conectado);
